Code Journal

Code Journal is a biweekly programming newsletter jointly run by Cprogramming.com and AI Horizon. Every newsletter contains four features: responses to three of the most interesting programming-related questions sent to Cprogramming.com and AI Horizon, an article by Alexander Allain, webmaster of Cprogramming.com, an article by Eric Suh, webmaster of AI Horizon, and a coding contest.

Back issues will be archived online in a searchable database. Code contest problem submissions will not be accepted past the deadline, unless there are no submissions. There is currently no prize for the coding contest; the top three programmers will be recognized in the next issue of Code Journal.
